## EOCS (Engine in CSharp)
> A simple open source 3D game engine built from scratch in C#.

Welcome to **EOCS**! This is a personal project aimed at creating a flexible and modular 3D game engine. The entire codebase is open-source, meaning you are free to download, modify, experiment with, and build the project yourself.

### About the Project
**EOCS** is being developed with a focus on:
* **Transparency:** Every line of code is accessible. No black boxes.

* **Learning & Collaboration:** Built to explore deep computer graphics concepts (OpenGL/Vulkan) and game architecture.

Whether you are interested in graphics programming, engine architecture, or just want to tinker with a custom C# engine, you are welcome here.

### Tech Stack
* **Language:** C# (.NET 8/9+)
* **Graphics API:** OpenGL (with potential Vulkan support in the future)
* **IDE:** Visual Studio Code / Visual Studio

### Installation & Build
1. Clone the repository: 
``` bash
git clone https://github.com/dovintc-off/Engine-on-CSharp.git
cd Engine-on-CSharp
```
2. Restore dependencies:
``` bash
dotnet restore
```
3. Change the code (Optional)
4. Build the project:
``` bash
dotnet run
```
